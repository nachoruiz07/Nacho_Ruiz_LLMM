El Modelo de Caja (Box Model)

Entendiendo cómo funcionan width, padding, border y margin.

Concepto Básico

Todo elemento en HTML es una caja rectangular. El modelo de caja consiste en:

  • Content: El contenido real (texto, imagen).
  • Padding: Espacio interior transparente alrededor del contenido.
  • Border: Borde que rodea el padding y el contenido.
  • Margin: Espacio exterior transparente fuera del borde (separa de otros elementos).

Comparación: content-box vs border-box

Caja 1: content-box

Width: 300px
Padding: 20px
Border: 10px

Ancho Total Real: 360px
(300 + 20*2 + 10*2)

Caja 2: border-box

Width: 300px
Padding: 20px
Border: 10px

Ancho Total Real: 300px
(El ancho incluye padding y border)

Esta caja tiene un margen grande para separarse de las de arriba.
