Introducción a CSS

Aprendiendo a dar estilo a nuestras páginas web.

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje que se usa para describir la presentación de un documento escrito en HTML.

Con CSS podemos controlar:

  • El color del texto.
  • El tipo de letra (fuentes).
  • El espaciado entre elementos.
  • El tamaño y posicionamiento.

Tipos de Selectores

Existen varios tipos de selectores en CSS:

  1. Selector de Etiqueta (ej. p, h1)
  2. Selector de Clase (ej. .mi-clase)
  3. Selector de ID (ej. #mi-id)

Recuerda: El ID debe ser único en toda la página, mientras que las clases se pueden reutilizar.
